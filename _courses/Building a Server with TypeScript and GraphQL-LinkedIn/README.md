## Differences Between GraphQL and REST

REST
• Multiple request to fetch data
• New changes to the API require versioning
• Runs risk of over-fetching / under-fetching data

GraphQL
• Single request to fetch data
• Schema can evolve without impacting clients
• More efficient data fetching
